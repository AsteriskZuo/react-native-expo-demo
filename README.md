# Introduction

实现 `expo` 和 `react-native` `native` 的源码一起编译运行。

# Create expo project

[Create expo project](./create.log)

# Build iOS native code

```sh
yarn add expo-dev-client
npx expo run:ios
```

[More detail](./create.log)

# Build Android native code

```sh
yarn add expo-dev-client
npx expo run:android
```

# Run app

```sh
yarn start
```
